# Spacing

---

This section defines spacing values used in the design system.

---

## **Spacing Tokens**

| Name      |  Value |
| --------- | -----: |
| `none`    |  `0px` |
| `default` |  `8px` |
| `xSmall`  |  `4px` |
| `small`   | `12px` |
| `medium`  | `16px` |
| `large`   | `24px` |
| `xLarge`  | `32px` |
| `xxLarge` | `40px` |

## **Example Usage**

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  container: {
    padding: 8, // default spacing
    marginBottom: 16, // medium spacing
  },
});
```
